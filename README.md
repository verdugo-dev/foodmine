# FoodMine

## Description

FoodMine is a full-stack food application that allows users to browse menus, place orders, and manage their favorite dishes. The frontend is built with Angular, providing a dynamic and responsive user interface, while the backend is powered by Express.js, handling API requests, user authentication, and data management.

## Features

- **Menu Browsing**: Explore a variety of food items with detailed descriptions and images.
- **User Authentication**: Secure login and registration system.
- **Order Management**: Place orders, view order history, and track status.
- **Favorites**: Save favorite dishes for quick access.
- **Responsive Design**: Optimized for desktop and mobile devices.

## Installation

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn
- [Any other prerequisites, like database setup]

### Frontend Setup

1. Navigate to the frontend directory:
   ```
   cd frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   ng serve
   ```

   The application will be available at `http://localhost:4200`.

### Backend Setup

1. Navigate to the backend directory (assuming it's separate):
   ```
   cd backend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables (create a `.env` file):
   ```
   PORT=3000
   DATABASE_URL=your_database_url
   JWT_SECRET=your_jwt_secret
   ```

4. Start the server:
   ```
   npm start
   ```

   The API will be available at `http://localhost:3000`.

## Usage

1. Start both frontend and backend servers as described above.
2. Open your browser and navigate to `http://localhost:4200`.
3. Register or log in to access the application features.
